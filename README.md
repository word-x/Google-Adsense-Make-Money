## 介绍
这个教程最早是相识的老站长的笔记，在小巴的强烈要求下，这个老站长今年重新整理了笔记并发布了这个最新版。
里面关于Adsense的心得，写得真的很好，有流程、有步骤、有技巧、有真实的收入截图和分析。
